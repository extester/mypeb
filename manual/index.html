peb function reference

peb — is an open-source php extension to run php as an Erlang cnode. It's similar to the traditional php mysql function , you can communicate with Erlang node easy.

  • peb_close() - Close Erlang node connection
  • peb_connect() - Open a connection to an Erlang node
  • peb_decode() - Decode an Erlang message
  • peb_encode() - Encode an Erlang message term
  • peb_errno() - Returns the numerical value of the error message from previous peb operation
  • peb_error() - Returns the text of the error message from previous peb operation
  • peb_linkinfo() - Get information about the link
  • peb_pconnect() - Open a persistent connection to an Erlang node
  • peb_receive() - receive an Erlang message from Erlang node
  • peb_send_byname() - Send an Erlang message by process name
  • peb_send_bypid() - Send an Erlang message by process ID
